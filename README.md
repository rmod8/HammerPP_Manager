# Hammer++ Manager
A program to allow the usage of Hammer++ with officially unsupported games, like Garry's Mod.

## Usage
[Video Tutorial for Garry's Mod](https://www.youtube.com/watch?v=N6a_9Ng8Tpw)

## Currently Supported Games
- Garry's Mod
- Fistful of Frags
